# 🍜 DO EAT Restaurant Website

A modern, responsive restaurant website developed for **DO EAT
Restaurant, Manchester**. It showcases the restaurant's menu, gallery,
restaurant information, customer reviews, contact details, and
reservation features with a clean, professional interface.

## 📖 Overview

This project was created to provide DO EAT with a professional online
presence where customers can:

-   Browse the restaurant menu
-   View featured dishes and gallery images
-   Learn about the restaurant
-   Read customer reviews
-   Find the restaurant location
-   Contact the restaurant
-   Make table reservations

## ✨ Features

-   Responsive landing page
-   Interactive navigation
-   About section
-   Complete menu section
-   Food & restaurant gallery
-   Why Choose Us section
-   Customer reviews
-   Location & contact information
-   Reservation form
-   Modern animations
-   Mobile-friendly responsive layout
